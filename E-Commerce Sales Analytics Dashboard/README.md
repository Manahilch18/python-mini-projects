# 🛒 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

The E-Commerce Sales Analytics Dashboard is a mini data analytics project developed using Python, Pandas, and NumPy. It analyzes sales data to generate useful business insights such as total sales, average order value, best-selling products, monthly sales trends, and customer performance.

This project demonstrates basic data analysis techniques and statistical calculations commonly used in real-world e-commerce businesses.

## 🚀 Features

* Calculate Total Sales Revenue
* Count Total Orders
* Find Average Order Value
* Identify Best-Selling Products
* Analyze Monthly Sales Trends
* Discover Top Customers
* Generate Sales Statistics using NumPy
* Interactive Dashboard using Streamlit

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Streamlit

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Manahilch18/E-Commerce-Sales-Analytics.git
```

2. Navigate to the project folder:

```bash
cd E-Commerce-Sales-Analytics
```

3. Install required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
streamlit run app.py
```
