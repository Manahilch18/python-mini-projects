# 🐍 Python Mini Projects

A collection of beginner-to-intermediate **Python mini projects** built using pure Python and standard libraries. Each project is self-contained, easy to run, and covers a different real-world use case.

### Prerequisites

- Python 3.x
- Some projects require extra libraries (noted per project below)

### Clone the Repository

```bash
git clone https://github.com/Manahilch18/python-mini-projects.git
cd python-mini-projects
```


---

## 💸 Expense Tracker

> `expense-tracker/expense_tracker.py`

An interactive CLI app to add, view, and categorize personal expenses with a running total.

**Run:**
```bash
python expense_tracker.py
```
**Tech:** Pure Python

---

## 🗂️ File Organizer

> `file-organizer/file_organizer.py`

Automatically sorts files in any folder into subfolders (Images, PDFs, Documents, Audio, Videos) by extension.

**Run:**
```bash
python file_organizer.py
```
**Tech:** Pure Python · `os` · `shutil`

---

## 🎯 Number Guessing Game

> `number-guessing-game/guessing_game.py`

Guess a random number between 1–50 within 5 attempts. Score points based on how few guesses you use.

**Run:**
```bash
python guessing_game.py
```
**Tech:** Pure Python · `random`

---

## 🔐 Password Analyzer

> `password-analyzer/password_analyzer.py`

Analyzes password strength across 5 criteria: length, uppercase, lowercase, digits, and special characters.

**Run:**
```bash
python password_analyzer.py
```
**Tech:** Pure Python

---

## 🎓 Student Grade Manager

> `student-grade-manager/grade_manager.py`

Add students, record subject marks, auto-calculate averages and letter grades, and save/load data.

**Run:**
```bash
python grade_manager.py
```
**Tech:** Pure Python · File I/O

---

## ⌨️ Typing Speed Tester

> `typing-speed-tester/typing_tester.py`

Measures your typing speed (WPM), mistake count, and accuracy against a randomly chosen sentence.

**Run:**
```bash
python typing_tester.py
```
**Tech:** Pure Python · `time` · `random`

---

## ✅ Smart To-Do Manager

> `smart-todo-manager/todo_manager.py`

A full-featured task manager with priorities, completion tracking, keyword search, and persistent storage.

**Run:**
```bash
python todo_manager.py
```
**Tech:** Pure Python · File I/O

---

## 🔗 URL Shortener

> `url-shortener/url_shortener.py`

Generate 6-character short codes for long URLs, retrieve originals, search, delete, and store locally.

**Run:**
```bash
python url_shortener.py
```
**Tech:** Pure Python · `random` · `string` · File I/O
